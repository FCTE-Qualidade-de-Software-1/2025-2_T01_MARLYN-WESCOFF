# Casos de Teste - Adequação Funcional

## Introdução
Esta página reúne todos os Casos de Teste derivados das métricas do GQM e definidos no plano de avaliação. Cada caso de teste é apresentado de forma estruturada, incluindo objetivo, pré-condições, procedimentos, dados necessários, critérios de aceitação e resultados esperados.

O conteúdo desta página tem como finalidade:
- Garantir rastreabilidade entre metas, questões, métricas e testes;
- Prover um conjunto de testes sistemático, padronizado e reproduzível;
- Documentar claramente como cada métrica será verificada durante a avaliação;
- Servir como referência central para a execução prática da Fase 4.

Todos os testes aqui descritos serão aplicados exclusivamente ao sistema Guardiões da Saúde, respeitando o escopo de segurança definido nas Fases 1 e 2.
